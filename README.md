# Django-Training-
Assignment: Event Booking System Admin Endpoints

1. View All Events
Endpoint: GET /admin/events
2. Add a New Event
Endpoint: POST /admin/events
3. Update Event Details
Endpoint: PUT /admin/events/<event_id>
4. Delete an Event
Endpoint: DELETE /admin/events/<event_id>
